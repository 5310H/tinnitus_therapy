Python Tone Generator

Generate calibrated therapy WAV files offline - no internet required.

Download tinnitus_generator.py

Requirements

  • Python 3.8+
  • NumPy - pip install numpy
  • SciPy - pip install scipy

Usage

# Generate 30 minutes of notch-filtered pink noise at 6000 Hz
python tinnitus_generator.py --type notch --freq 6000 --duration 1800

# Generate decorrelated white noise
python tinnitus_generator.py --type decorrelated --color white --duration 900

# Generate a pure tone for pitch-matching
python tinnitus_generator.py --type tone --freq 4500 --duration 30

Options

  • --type - notch, decorrelated, or tone
  • --freq - Center frequency in Hz (default: 6000)
  • --color - Noise color: white, pink, brown (default: pink)
  • --duration - Duration in seconds (default: 600)
  • --notch-width - Notch width in octaves (default: 1.0)
  • --sample-rate - Sample rate in Hz (default: 44100)
  • --output - Output filename (auto-generated if omitted)

Output

Files are saved as 16-bit stereo WAV. Decorrelated mode generates independent noise per channel. Notch mode applies a bandstop filter centered on your tinnitus frequency.

Safety

Always start at low volume. Therapy sounds should be played at or slightly below your tinnitus perception level (mixing point). Consult an audiologist for personalized guidance.
